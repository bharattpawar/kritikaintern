# AI Usage Notes

## LLM Selection

**Chosen Provider**: Groq API  
**Model**: Llama 3.1 70B Versatile

### Why Groq?

1. **Free Tier**: 14,400 requests/day with no credit card required
2. **Speed**: ~300 tokens/second - fastest free option available
3. **Code Understanding**: Llama 3.1 70B excels at code comprehension
4. **Reliability**: Stable API with 99%+ uptime
5. **Cost**: Completely free for this use case

### Alternatives Evaluated

- **OpenAI GPT-4**: Excellent but requires payment
- **Anthropic Claude**: Great for code but paid
- **Hugging Face**: Free but slower and less reliable
- **OpenRouter**: Good aggregator but Groq is faster

## Key Learnings

1. AI is great for boilerplate but needs review
2. All generated code was tested and verified
3. UI/UX requires human judgment
4. Integration work is primarily manual
5. Understanding the code is critical

## Transparency

- Reviewed for correctness

- Modified to fit requirements

AI accelerated development but didn't replace critical thinking, problem-solving, or technical understanding.
